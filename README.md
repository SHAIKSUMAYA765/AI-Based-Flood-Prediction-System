# 🌊 AI-Based Flood Prediction System

## 📌 Overview

The **AI-Based Flood Prediction System** is a Machine Learning-powered web application that predicts flood risk using environmental and weather parameters. The system analyzes factors such as temperature, humidity, cloud cover, and rainfall to estimate the likelihood of flooding. It provides users with a simple, interactive interface for quick and intelligent flood risk assessment.

---

## 🎯 Project Objective

The objective of this project is to demonstrate the practical application of Artificial Intelligence and Machine Learning in disaster management by providing an intelligent flood prediction system that supports awareness and preparedness.

---

## ✨ Features

* 🌊 AI-Based Flood Prediction
* 🤖 Machine Learning using XGBoost
* 📊 Weather Parameter Analysis
* ⚡ Instant Prediction Results
* 🎨 Modern and Responsive User Interface
* 🌐 Flask-Based Web Application

## 🚀 Installation

### Clone the repository

```bash
git clone <https://github.com/SHAIKSUMAYA765/AI-Based-Flood-Prediction-System>
```

### Navigate to the project folder

```bash
cd AI-Based-Flood-Prediction-System
```

### Install the required packages

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open your browser

```text
http://127.0.0.1:5000
```
